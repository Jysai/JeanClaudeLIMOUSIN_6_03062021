Dans le terminal, 
A partir du dossier backend, entrez la commande "npm install" puis "node server".

Vous devriez voir ce message: 
"Listening on port 3000
Connexion à MongoDB réussie !"

A partir du dossier frontend, entrez la commande "npm run start"

Dans votre navigateur, entrez l'url suivante => http://localhost:8070
